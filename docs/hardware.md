# Hardware

This document will contain the complete hardware specification for the Mobile Robotic Manipulator.

## Current Development Hardware

- NodeMCU-32S ESP32 development board
- MPU-compatible inertial measurement unit
- Six-wheel UGV platform
- Six-DOF robotic arm
- PCA9685 servo driver
- DC motor drivers
- Servo motors
- Battery and voltage regulators
- Buzzer
- Headlight

Exact models, ratings and quantities will be documented after system integration.
