# Wiring Documentation

Complete wiring diagrams for the Mobile Robotic Manipulator will be added as hardware integration progresses.

## Currently Tested Connection

| Device | Signal | ESP32 Pin |
|---|---|---|
| MPU-compatible IMU | SDA | GPIO 21 |
| MPU-compatible IMU | SCL | GPIO 22 |
| MPU-compatible IMU | VCC | 3.3V |
| MPU-compatible IMU | GND | GND |
